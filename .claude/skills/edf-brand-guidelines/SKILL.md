# Charte Graphique Groupe EDF — Skill de Référence

## Usage de ce skill

Ce skill couvre TOUTE la charte EDF 2021. Pour chaque création :

1. Identifier le **support** (print / digital / web / social / signalétique…)
2. Identifier la **famille de marque** (EDF groupe ou filiale)
3. Lire les sections pertinentes via les fichiers de référence
4. Appliquer les règles dans l'ordre : Couleurs → Logo → Cartouche → Typographie → Composition

**Fichiers de référence à lire selon le besoin :**
- `references/01-brand-platform.md` — Raison d'être, valeurs, plateforme de marque
- `references/02-colors.md` — Palette complète, codes hex/RGB/CMJN/PMS, web, dégradés, effet lumineux
- `references/03-logo.md` — Logo, espace de protection, tailles, applications, interdits
- `references/04-typography.md` — Polices EDF 2020, Work Sans, Arial, usages, mise en exergue
- `references/05-cartouche.md` — Cartouche carré, tailles, placement, couleurs, bichromie, transparence
- `references/06-iconography.md` — Principes photo, dominantes couleur, pictogrammes, illustrations
- `references/07-patterns-symbols.md` — Motif baguette, symboles +/−/>, applications
- `references/08-composition.md` — Mises en page verticales, réseaux sociaux, bonnes pratiques
- `references/09-digital-web.md` — Règles spécifiques web, accessibilité, app mobile, couleurs web
- `references/10-applications.md` — Exemples : affiches, PPT, newsletter, édition, goodies

## Synthèse des règles absolues (à mémoriser)

### ✅ Règles TOUJOURS applicables

| Règle | Détail |
|-------|--------|
| **Logo immuable** | Formes et couleurs non modifiables, jamais en noir, jamais déformé |
| **Logo position** | Toujours en haut à gauche, à 2F du bord (print) ou 1F (digital restreint) |
| **Camaïeu unique** | Un seul camaïeu (orange, bleu OU vert) par support |
| **Cartouche carré** | Toujours carré, jamais rectangulaire |
| **Logo + texte** | Le cartouche ne peut pas vivre avec un logotype seul |
| **Transparence max** | Transparence du cartouche à 10% maximum |
| **Bichromie** | 1/3 blanc (logo) + 2/3 couleur (texte), jamais en transparence |
| **Pictogrammes** | Maximum 3 pictos par support |
| **Symboles** | Maximum 1 symbole par composition |

### ❌ Interdits absolus

- Logo en noir, en monochrome couleur, vertical, déformé
- Turbine seule sans le texte "edf"
- Logo en couleurs sur fond coloré ou visuel complexe
- Logo blanc sur visuel clair
- Mélange de camaïeux sur un même support (sauf bleu+orange ou bleu+vert en touches)
- Cartouche bicolore sur le motif
- Plus d'un symbole par composition

## Architecture de marque

Le Groupe EDF s'exprime d'une seule voix avec ses filiales.
Toutes les filiales partagent la même turbine EDF, leur nom propre, et la mention "GROUPE EDF".

**Règle filiales :** L'espace de protection est défini par la **pale gauche de la turbine** (≠ "F" pour EDF).
